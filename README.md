
## 介绍

这个项目是基于unisat-wallet-sdk的二次开发
主要实现功能：

```shell
1：钱包创建：通过 mnemonic: 恢复钱包，通过 privateKey: 恢复钱包，随机生成钱包
2: 转账： btc转账， brc20转账，runes转账
3：签名：
4：查询：基于unisat 的api

```


## 项目使用yarn管理依赖


```shell
# install
yarn install

# build
yarn build:esm

# test
yarn test
```

## 使用技术

```shell
0:yarn 依赖包管理工具
1：gulp 构建工具
2：webpack 打包工具
3: eslint 代码规范工具
4: babel 代码转换工具
5: tsconfig.json ts配置文件
6: bitcoinjs bitcoinjs-lib的二次开发

```
